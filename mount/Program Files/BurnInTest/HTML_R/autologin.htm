Auto-login

    

You can use Rebooter to set and clear the Windows Auto-login feature. By entering your Domain, User name and password then clicking on ‘Set Auto-login”, you can avoid the login prompt appearing during the windows boot process.

The feature is only available with Windows NT, 2000 and XP.

Although enabling autologon can make it more convenient to use Windows, use of this feature is a security risk. See the FAQ for more details.
