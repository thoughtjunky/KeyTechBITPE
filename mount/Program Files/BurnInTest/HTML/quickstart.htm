Quick start – How to start testing

    

 

 

Here’s a summary of the steps to go through to start testing.
 
1) Use the Configuration, Test Duty Cycles window to select the type of tests you wish to perform.
 
2) Use the Configuration, Test Preferences window to set any parameters that you wish to use, e.g.  which hard drive to use.
 
3) Put paper in the printer, a data DVD in the DVD-drive and connect any loopback plugs (if you selected these tests).
 
4) Click on the Start button.

That’s it.
