Parallel ports

    

The Parallel port test is only included in the BurnInTest Pro version only.

Parallel port settings

Parallel port

The communications port that will be used by the parallel port test, e.g. LPT1, LPT2 or LPT3 (only ports currently installed will be displayed). See the parallel port test for more details.

Only a single parallel port may be tested at one time.

Detection and loopback test / Port detection only

It is possible to select either a “Detect and loopback test” or a “Detect only” test. The, detect and loopback test, requires the use of parallel port loop back plug. The, detect only, test does not require a loopback plug. When possible use the loopback test as it is a more though test.
