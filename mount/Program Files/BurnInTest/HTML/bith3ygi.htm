Test Preferences – Errors

Action on Error

When BurnInTest detects an error, three options are available.

Continue. The error counter is incremented and testing continues.

Continue and beep. The error counter is incremented, the computer beeps and testing continues.

Stop all tests. All windows (except the main window) are closed and all the tests are stopped. The failed window is then displayed.

Windows errors (Abort / Retry / Cancel)

You can specify whether BurnInTest should block or allow critical Windows errors.
